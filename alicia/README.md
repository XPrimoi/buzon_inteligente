Neste directorio atópase parte das implementacións das prácticas P2 e P3, cada unha co seu código e o seu README para explicalo.

En [P2](https://github.com/XPrimoi/buzon_inteligente/tree/main/alicia/p2) abórdase a implementación dun nodo ESP32 que emite e recibe sinais IR e comunica eventos e lecturas mediante MQTT, permitindo contar cartas introducidas desde FlowFuse e actualizar o estado do buzón intelixente.

En [P3](https://github.com/XPrimoi/buzon_inteligente/tree/main/alicia/p3) desenvólvese a integración do sensor de presión, colisión e IR, unificando o seu código e controlándoos dende un dashboard en Node‑RED despregado en Cloudlet.



>This directory contains part of the implementations for practices P2 and P3, each with its code and its README explaining the work.
>
>In [P2](https://github.com/XPrimoi/buzon_inteligente/tree/main/alicia/p2) the implementation focuses on an ESP32 node that sends and receives IR signals and communicates events and readings via MQTT, enabling letter counting from FlowFuse and updating the smart mailbox status.
>
>In [P3](https://github.com/XPrimoi/buzon_inteligente/tree/main/alicia/p3) the pressure, collision and IR sensors are integrated, unifying their code and controlling them through a Node‑RED dashboard deployed on Cloudlet.
